# NCT Gateway

Это сервис проверяет поступающие запросы и проксирует их на другие сервисы.

## v0.1.0

В текущей версии реализована базовая проверка по ip адрессу и проксирование запросов.

## Запуск

### Установка

```shell
uv sync
. ./.venv/bin/activate
. ./.venv/Scripts/activate # для Windows
```
